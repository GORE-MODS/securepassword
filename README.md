# securepassword
A command for linux that generates a random secure password.
# how to make it a command:
# No1 go into download by doing: cd /home/youruser/Downloads
# No2 put it in /usr/local/bin by doing this command: sudo cp securepassword /usr/local/bin
# No3 (FINAL) run this command to allow you to run it: sudo chmod +x /usr/local/bin/securepassword
# Your done! now you have securepassword
